# Omahi

_O mahi, love every phase._

A browser extension that plans your schedule, meals, meetings, workouts, big tasks, and rest around your menstrual cycle's four phases. Local-only — no server, no account, your data never leaves your browser.

> All food and workout guidance is lifestyle suggestion, not medical advice.

## Development

```sh
pnpm install
pnpm dev        # WXT dev server with HMR
pnpm test       # unit tests (Vitest)
pnpm build      # production build
pnpm e2e        # Playwright e2e (build first)
```

Repo layout, conventions, and testing strategy: see [CLAUDE.md](./CLAUDE.md).
